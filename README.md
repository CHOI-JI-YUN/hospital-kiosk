# 🏥 병원 키오스크 시스템
C# WinForms 기반 접수 및 대기 관리 프로그램

---

## 📌 프로젝트 소개
본 프로젝트는 병원 키오스크 환경을 가정하여, 사용자가 직접 접수를 진행하고 진료과를 선택한 뒤 대기 및 호출 상태를 확인할 수 있도록 구현한 프로그램입니다.  
실제 사용자 흐름을 고려하여 직관적인 UI와 효율적인 대기 관리 기능을 중심으로 설계했습니다.

---

## 🔄 프로젝트 흐름
1. 사용자가 메인 화면에서 접수 / 수납 기능 선택
2. 신규 / 기존 환자 여부에 따라 접수 진행
3. 진료과 및 증상 선택
4. 대기 목록에 등록 후 호출 상태 확인
5. 진료 후 처방전 화면 출력

---

## 🛠 개발 환경
- Language: C#
- Framework: WinForms
- IDE: Visual Studio

---

## 💡 주요 기능
- 신규 / 기존 환자 구분 접수
- 진료과 및 증상 선택 기능
- 대기 인원 및 대기 순서 표시
- 환자 호출 및 진료 상태 표시
- 처방전 출력 화면 제공

---

## 👨‍💻 구현 내용
- Dictionary를 활용한 환자 정보 관리
- Queue 자료구조를 이용한 대기열 관리 시스템 구현
- DataGridView를 활용한 대기 목록 UI 구성
- 버튼 이벤트 기반 사용자 인터랙션 처리
- Form 간 전환을 통한 기능 분리 (접수 / 대기 / 처방전)

---

## 📁 파일 구성
- `Form1.cs` : 메인 키오스크 화면 로직
- `Form1.Designer.cs` : 메인 화면 UI 구성
- `FormPrescription.cs` : 처방전 화면 로직
- `Program.cs` : 프로그램 실행 진입점
- `WaitItem.cs` : 대기 데이터 관리 클래스
